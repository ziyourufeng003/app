# Flutter 开发环境搭建指南（Windows）

> 适用于从零开始搭建 Flutter 开发环境，目标项目：简谱工具箱（d:\app）

---

## 仓库地址

- **GitHub**：https://github.com/ziyourufeng003/app
- **克隆命令**：`git clone https://github.com/ziyourufeng003/app.git d:\app`

环境搭建完成后，直接克隆项目即可获取最新代码，无需手动创建项目。

---

## 第一步：安装 Chocolatey（包管理器）

以**管理员身份**打开 PowerShell，执行：

```powershell
Set-ExecutionPolicy Bypass -Scope Process -Force; [System.Net.ServicePointManager]::SecurityProtocol = [System.Net.ServicePointManager]::SecurityProtocol -bor 3072; iex ((New-Object System.Net.WebClient).DownloadString('https://community.chocolatey.org/install.ps1'))
```

安装完成后关闭 PowerShell，重新以管理员身份打开。

---

## 第二步：安装 Flutter SDK

```powershell
choco install flutter -y
```

---

## 第三步：配置国内镜像环境变量（用户级永久生效）

```powershell
[System.Environment]::SetEnvironmentVariable('PUB_HOSTED_URL', 'https://pub.flutter-io.cn', 'User')
[System.Environment]::SetEnvironmentVariable('FLUTTER_STORAGE_BASE_URL', 'https://storage.flutter-io.cn', 'User')
```

---

## 第四步：安装 Android Studio

```powershell
choco install android-studio -y
```

安装完成后设置 JAVA_HOME：

```powershell
[System.Environment]::SetEnvironmentVariable('JAVA_HOME', 'C:\Program Files\Android\Android Studio\jbr', 'User')
```

---

## 第五步：安装 Android SDK

```powershell
choco install android-sdk -y
```

配置 Flutter 识别 Android SDK 路径：

```powershell
flutter config --android-sdk "C:\Android\android-sdk"
```

---

## 第六步：安装 Android SDK 组件（platform-36 + build-tools-36）

### 方法一：sdkmanager（网络通畅时）

```powershell
$env:JAVA_HOME='C:\Program Files\Android\Android Studio\jbr'
C:\Android\android-sdk\cmdline-tools\latest\bin\sdkmanager.bat "platforms;android-36" "build-tools;36.0.0"
```

### 方法二：手动下载（网络不通时）

1. 创建 `C:\Android\android-sdk\repositories.cfg`，内容：

```xml
<?xml version="1.0" encoding="UTF-8"?>
<site>
  <repository>
    <url>https://mirrors.cloud.tencent.com/AndroidSDK/</url>
    <enabled>true</enabled>
  </repository>
</site>
```

2. 下载 commandlinetools（需要 Python）：

```powershell
python -c "import requests; r=requests.get('https://mirrors.cloud.tencent.com/AndroidSDK/commandlinetools-win_11076708_latest.zip', proxies={'https':'http://127.0.0.1:7897'}, verify=False); open('C:/Android/cmdline-tools.zip','wb').write(r.content)"
```

解压到 `C:\Android\android-sdk\cmdline-tools\latest`（注意：解压后如果嵌套了一层目录，需要把内层内容移到 latest 下）

3. 下载 platform-36 和 build-tools-36：

```powershell
python -c "import requests; r=requests.get('https://mirrors.cloud.tencent.com/AndroidSDK/platforms/android-36.zip', proxies={'https':'http://127.0.0.1:7897'}, verify=False); open('C:/Android/platform-36.zip','wb').write(r.content)"

python -c "import requests; r=requests.get('https://mirrors.cloud.tencent.com/AndroidSDK/build-tools_r36_windows.zip', proxies={'https':'http://127.0.0.1:7897'}, verify=False); open('C:/Android/build-tools-36.zip','wb').write(r.content)"
```

- `platform-36.zip` 解压到 `C:\Android\android-sdk\platforms\android-36`
- `build-tools-36.zip` 解压到 `C:\Android\android-sdk\build-tools\36.0.0`

**注意**：
- 如果不用代理，去掉 `proxies={'https':'http://127.0.0.1:7897'}` 参数
- 如果代理端口不是 7897，改成实际端口
- 解压后检查是否嵌套了多余目录，如有需要把内层文件移到正确位置

## 第七步：安装 Visual Studio 2022（Windows 桌面开发需要）

```powershell
choco install visualstudio2022community -y
choco install visualstudio2022-workload-nativecoredesktool -y
```

或者手动安装 Visual Studio Community 2022，勾选 **"使用C++的桌面开发"** 工作负载。

---

## 第八步：接受 Android 许可证

```powershell
$env:JAVA_HOME='C:\Program Files\Android\Android Studio\jbr'; "y`ny`ny`ny`ny`n" | flutter doctor --android-licenses
```

---

## 第九步：验证环境

**关闭当前终端，重新打开新终端**（让环境变量生效），执行：

```powershell
flutter doctor
```

全部通过应显示：`No issues found!`

预期结果：
- [√] Flutter
- [√] Windows Version
- [√] Android toolchain
- [√] Chrome
- [√] Visual Studio
- [√] Connected device
- [√] Network resources

---

## 第十步：获取项目代码并运行

```powershell
git clone https://github.com/ziyourufeng003/app.git d:\app
cd d:\app
flutter pub get
flutter run -d windows
```
